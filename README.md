# 🚗 Car Price Prediction

## 📌 Overview

A machine learning project to predict car selling prices based on features like present price, mileage, fuel type, and car age.

---

## 🧠 Model

* Algorithm: Random Forest Regressor
* Task: Regression

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
* Streamlit

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
python main.py
streamlit run app/app.py
```

---

## 📊 Features

* Data preprocessing & feature engineering
* Model training & evaluation
* Web app using Streamlit
